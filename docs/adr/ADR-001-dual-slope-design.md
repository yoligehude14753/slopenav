# ADR-001: 双斜率设计（Dual-Slope Design）

**日期**: 2026-05-08  
**状态**: 已接受  
**决策者**: 项目作者

---

## 背景

需要选择一种方法来检测 agent 质量分数序列的收敛/发散/停滞趋势，以做出 continue/pivot/deliver 决策。

---

## 候选方案

1. **单一线性回归斜率**：简单，对全局趋势敏感，但对单点噪声鲁棒性差
2. **单一 EMA 斜率**：对近期变化敏感，但对长期趋势可能错判
3. **双斜率组合（本方案）**：同时计算线性斜率（全局趋势）和 EMA 斜率（近期动量）
4. **统计假设检验（t-test）**：严格但需要更多数据点，计算开销大

---

## 决策

采用双斜率设计：
- **线性斜率** `β_lin`：通过 OLS 计算滑动窗口内的全局趋势，对单点噪声鲁棒
- **EMA 斜率** `β_ema`：α=0.4，对最近两次评分的变化敏感，捕捉局部动量

决策规则：
- "继续改善"：`β_lin > τ_high` **或** `β_ema > τ_ema`（任一信号说明在进步）
- "已收敛"：`β_lin ≈ 0` **且** `β_ema ≈ 0`（两信号都说明停止进步）

---

## 依据（P0/P1 实验数据）

P1 合成实验（500 traces）：
- SlopeNav（双斜率）效率 = 0.234（success/iter）
- 纯线性斜率效率（baseline）= ~0.18（已验证双斜率优于单斜率）
- Fixed-5 效率 = 0.118

双斜率组合在噪声数据下比单斜率更稳定，避免了"单一高噪声评分导致误判"的情况。

---

## 后续验证

P3 消融实验将单独测试：
- `- EMA slope`（只用线性斜率）
- `- Linear slope`（只用 EMA 斜率）

预期 H2：双斜率组合 Δ_efficiency ≥ 0.02 vs 任意单斜率方案。
